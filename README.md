# Hello, I'm Angel

Modern 3D interactive portfolio website for Angel Medina

## Tech Stack

**Frontend**
- Next.js 15 + TypeScript
- React Three Fiber + Three.js
- Tailwind CSS
- GSAP + Framer Motion

**Backend**
- Rust + Axum
- PostgreSQL
- SQLx

**Deployment**
- Vercel (Frontend)
- Fly.io (Backend)
- Domain: angelmedina.io

## Color Scheme

- Black: #000000
- Forest Green: #228B22
- Crimson Maroon: #800020
- Light Gray: #D3D3D3
